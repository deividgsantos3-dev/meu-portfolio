Deep Web & Dark Web

Abaixo da superfície: Os riscos de infraestrutura, anonimato comprometido e exposição de hardware em redes não indexadas.

Traffic Leak

Malicious Exit Nodes (Nós de Saída)

Ao usar o Tor, seu tráfego passa por voluntários. Um "nó de saída" malicioso pode ler tudo o que você envia se o site não tiver HTTPS.

VETOR: Traffic Sniffing em Exit Relay.
RISCO: Captura de credenciais e dados em texto plano.
Data Mining

VPNs "Grátis" para Deep Web

Navegadores que prometem acesso fácil com "VPN Integrada" costumam ser Honey Pots (armadilhas) que coletam seu IP real e vendem seus dados.

VETOR: Browser Log Fingerprinting.
RISCO: Exposição do seu IP real e localização.
Tracking

Browser Fingerprinting

Sites na Dark Web usam scripts avançados para identificar a resolução da sua tela, fontes instaladas e hardware, criando um ID único seu.

VETOR: Canvas/WebGL Identification.
RISCO: Desanonimização persistente do seu computador.
Critical

JavaScript Exploits (Zero-Day)

Muitos sites .onion carregam scripts que exploram falhas no motor do navegador para executar código direto no seu sistema operacional.

VETOR: Remote Code Execution (RCE).
RISCO: Controle total do seu PC pelo atacante.
Financial

Marketplace Scams & Phishing

90% dos fóruns de venda na Dark Web são clones criados apenas para roubar Bitcoins de usuários curiosos.

VETOR: Crypto Wallet Hijacking.
RISCO: Perda financeira irreversível (Sem estorno).
Legal Risk

Honey Pots (Autoridades)

Fóruns mantidos por agências de inteligência para monitorar quem acessa conteúdos ilegais, rastreando usuários por meses.

VETOR: Long-term Metadata Analysis.
RISCO: Implicação em investigações criminais.
Privacy

Acesso ao Periférico

Scripts que tentam ativar sua webcam ou microfone silenciosamente assim que um link .onion é aberto no navegador sem proteção.

VETOR: Peripheral Plugin Exploit.
RISCO: Chantagem e exposição de imagens íntimas.
Metadata

Leaking via Arquivos

Baixar ou subir documentos em fóruns obscuros sem limpar metadados pode revelar o nome do seu usuário do Windows e o ID do seu HD.

VETOR: EXIF/NTFS Metadata Leak.
RISCO: Identificação física do autor do arquivo.
Infection

Falsos Visualizadores de PDF

Documentos que exigem download de um "visualizador especial" para serem lidos, que na verdade é um trojan de acesso remoto (RAT).

VETOR: Dropper Malware.
RISCO: Espionagem contínua via Keylogger.
Leak

IPv6 Leakage

Muitas configurações de VPN ou Tor falham em mascarar o IPv6, deixando seu endereço de rede real visível para o servidor .onion.

VETOR: Improper Tunnel Configuration.
RISCO: Quebra total do túnel de anonimato.
REF: #PF-MONITOR CRITICAL

Nós de Saída Controlados (PF/FBI)

Agências de segurança operam milhares de nós do Tor para interceptar tráfego e identificar a origem de usuários.

VETOR: De-anonymization Relay.
ALVO: IP Real do Usuário.
REF: #HONEY-POT HIGH

Fóruns Honey Pot

Sites de serviços ilegais criados pela polícia para coletar logs de quem entra, o que busca e o que escreve.

VETOR: Database Logging.
ALVO: Perfilamento criminal.
REF: #NIT-ATTACK CRITICAL

Network Investigative Techniques

Malwares enviados via navegador pela polícia que forçam o seu computador a enviar o seu IP real para um servidor central.

VETOR: Government Malware (NIT).
ALVO: Localização Física.
REF: #HW-ID MEDIUM

Extração de MAC Address

Scripts em sites .onion que exploram falhas de áudio/vídeo para ler o endereço físico da sua placa de rede.

VETOR: Hardware Leaking.
ALVO: Identificação única do PC.
REF: #CORRELATION HIGH

Ataques de Correlação

Monitoramento do tempo de entrada e saída de dados no seu roteador doméstico para bater com o tráfego na Dark Web.

VETOR: Traffic Analysis.
ALVO: Quebra de anonimato Tor.
REF: #JS-EXPLOIT CRITICAL

Exploits de JavaScript Ativos

Navegar com JS ativado permite que sites executem comandos que revelam o nome do seu usuário no Windows.

VETOR: RCE (Remote Code Execution).
ALVO: Arquivos do Sistema Operacional.
REF: #VPN-LOGS HIGH

VPNs com "Backdoor"

Muitas VPNs que prometem anonimato têm acordos de cooperação com a Polícia Federal para entregar logs em tempo real.

VETOR: Log Handover.
ALVO: Histórico de navegação.
REF: #METADATA MEDIUM

Vazamento por Metadados de Imagem

Ao postar ou enviar uma foto, metadados (EXIF) podem revelar a marca do celular e coordenadas GPS exatas.

VETOR: EXIF Data Leak.
ALVO: Geolocalização doméstica.
REF: #BROWSER-FP MEDIUM

Impressão Digital do Navegador

A combinação da sua versão de sistema, fontes e resolução cria uma assinatura que te persegue em todos os sites.

VETOR: Canvas Fingerprinting.
ALVO: Rastreamento persistente.
REF: #CRYPTO-TRACE HIGH

Rastreamento de Blockchain

Empresas de segurança analisam transações de Bitcoin para ligar carteiras anônimas a contas de corretoras (KYC).

VETOR: Ledger Analysis.
ALVO: Identidade financeira.
REF: #DNS-LEAK MEDIUM

Vazamento de DNS

Mesmo no Tor, seu computador pode tentar resolver endereços via DNS da sua operadora (Vivo/Claro), revelando o site visitado.

VETOR: DNS Leakage.
ALVO: Logs da Operadora.
REF: #PHISHING-CLONE HIGH

Clones de Mercados Ocultos

Versões falsas de sites famosos que roubam sua chave privada e senhas assim que você tenta logar.

VETOR: Credential Harvesting.
ALVO: Acesso a contas e fundos.
REF: #VM-ESCAPE CRITICAL

Fuga de Máquina Virtual (VM Escape)

Malwares avançados que conseguem "pular" da VM infectada para o seu computador real (Host).

VETOR: Hypervisor Exploit.
ALVO: Sistema Host e Rede Local.
REF: #WEB-RTC MEDIUM

Vulnerabilidade WebRTC

Protocolos de chat de vídeo que podem vazar o seu IP local (192.168.x.x) mesmo atrás de uma VPN.

VETOR: STUN/TURN Request.
ALVO: Estrutura de rede interna.
REF: #FILE-TRIGGER HIGH

Arquivos "Beacon" (PDF/DOCX)

Arquivos que, ao serem abertos *offline*, tentam fazer uma conexão externa para avisar ao dono onde você está.

VETOR: External Resource Injection.
ALVO: IP e Horário de acesso.
REF: #CSS-TRACKING LOW

Rastreamento via CSS

Uso de folhas de estilo para carregar imagens invisíveis que confirmam o seu tamanho de tela e tipo de dispositivo.

VETOR: CSS Injection.
ALVO: Perfilamento técnico.
REF: #STYLOMETRY HIGH

Análise de Estilometria

Sistemas de IA que analisam o seu jeito de escrever para comparar com seus posts em redes sociais abertas.

VETOR: Linguistic Pattern Matching.
ALVO: Identidade real do autor.
REF: #ISP-BLOCK MEDIUM

Monitoramento de Início de Fluxo Tor

Sua operadora sabe que você está usando o Tor. Isso por si só pode colocar seu IP em uma lista de monitoramento.

VETOR: Deep Packet Inspection (DPI).
ALVO: Flag de "Usuário Suspeito".
REF: #TIME-ANALYSIS MEDIUM

Análise de Fuso Horário

Scripts que checam a diferença entre o relógio do seu sistema e o relógio do servidor para estimar seu país.

VETOR: Clock Skew / Timezone Leak.
ALVO: Localização geográfica.
REF: #HIDDEN-SERVICE CRITICAL

Vulnerabilidades de Servidores .onion

Se você hospeda um site, falhas no Apache/Nginx podem revelar o IP do servidor, levando a PF direto ao seu endereço.

VETOR: Server-Side Misconfiguration.
ALVO: Localização do servidor físico.
