Sistemas Paralelos e Terceiros

Análise de vetores de intrusão via dispositivos de monitoramento, automação predial e serviços externos.

ID: #01 CRITICAL

Câmeras IP (DVR/NVR)

Equipamentos de CFTV com senhas padrão (admin/1234) e firmwares desatualizados que permitem acesso via Telnet/HTTP.

EXPLOIT: Remote Code Execution (RCE)
ACCESS: Pivot to Internal Network
ID: #02 HIGH

Controle de Acesso Biométrico

Terminais de ponto e biometria que armazenam credenciais em bancos de dados SQL sem criptografia.

DATABASE: SQLite / Access
VULN: Credential Extraction
ID: #03 CRITICAL

Ar-Condicionado / HVAC

Sistemas de refrigeração industrial conectados à rede para gestão remota via protocolos inseguros (BACnet/Modbus).

PROTO: BACnet (UDP 47808)
VULN: Unauthenticated Control
ID: #04 HIGH

No-Breaks (UPS) Inteligentes

Placas de gerenciamento SNMP em UPS que permitem desligar servidores remotamente se a senha 'public' estiver ativa.

SERVICE: SNMP v1/v2c
ATTACK: Denial of Service (DoS)
ID: #05 MEDIUM

Smart TVs em Salas de Reunião

Televisores com microfones e câmeras ativos, conectados ao Wi-Fi corporativo com vulnerabilidades de navegador.

VECTOR: Browser Exploit
RISK: Corporate Espionage
ID: #06 CRITICAL

Gateways de Automação (IoT)

Hubs que conectam sensores de iluminação e presença, muitas vezes instalados por empresas de engenharia elétrica sem auditoria de TI.

OS: Embedded Linux (Old Kernel)
ACCESS: Persistence (Backdoor)
ID: #07 HIGH

Impressoras de Crachás

Dispositivos de terceiros que costumam ter interfaces web abertas e permitem o download de bancos de dados de funcionários.

VULN: Directory Traversal
DATA: PII (Personal Info) Leak
ID: #08 HIGH

Sensores de Incêndio (Fire Alarm)

Painéis de alarme que integram com a rede para avisos externos, mas que podem ser usados para movimentação lateral.

INTERFACE: Serial-to-Ethernet
ATTACK: Network Sniffing
ID: #09 CRITICAL

VPN de Suporte do Fabricante

Túneis VPN permanentes deixados abertos para que o fornecedor de máquinas industriais faça manutenção remota.

TYPE: Unmonitored Site-to-Site
RISK: Third-Party Compromise
ID: #10 MEDIUM

Totens de Autoatendimento

Quiosques para visitantes que rodam Windows e permitem o "Escape" da aplicação para acessar o CMD do sistema.

VECTOR: Kiosk Breakout
PRIV: Local Administrator
ID: #11 HIGH

Máquinas de Café / Vending Machines

Dispositivos IoT conectados para telemetria de estoque que servem de entrada para scanners de rede.

NETWORK: Guest Wi-Fi Bridge
TOOL: Port Scanning
ID: #12 CRITICAL

Controladores Lógicos (PLC)

Sistemas em linhas de produção que não exigem autenticação para alteração de parâmetros críticos.

PROTO: Modbus TCP (502)
IMPACT: Physical Sabotage
ID: #13 HIGH

Telefonia IP (VoIP)

Aparelhos de telefone em mesas que podem sofrer interceptação de áudio se o protocolo SIP não for criptografado.

ATTACK: Eavesdropping
DECODING: RTP Stream Capture
ID: #14 MEDIUM

Relógios Digitais de Rede (NTP)

Displays de hora sincronizados que utilizam versões vulneráveis de serviços de rede.

CVE: NTP Amplification
USE: DDoS Reflection
ID: #15 CRITICAL

Geradores de Energia (Backups)

Sistemas de monitoramento de combustível e status de geradores acessíveis via web sem HTTPS.

VULN: Clear-text Credentials
IMPACT: Infrastructure Shutdown
ID: #16 HIGH

Elevadores Inteligentes

Sistemas de chamada de elevadores integrados ao controle de acesso que utilizam APIs mal protegidas.

VECTOR: API Manipulation
RESULT: Floor Access Bypass
ID: #17 MEDIUM

Painéis Solares / Inversores

Gestão de energia fotovoltaica com dashboards na nuvem que, se invadidos, revelam a topologia da rede local.

DATA: Network Topology Leak
RISK: Cloud-to-Ground Pivot
ID: #18 HIGH

Sistemas de Som / Paging

Auto-falantes IP usados para anúncios que podem ser hackeados para emitir áudios falsos ou pânico.

TYPE: Unauthenticated Audio Inject
IMPACT: Social Engineering
ID: #19 CRITICAL

Pontos de Carregamento EV

Estações de recarga de carros elétricos no estacionamento conectadas à extranet da empresa.

PROTO: OCPP
VULN: Lack of Encryption
ID: #20 HIGH

Frigobares Inteligentes (Executivos)

Dispositivos de luxo em salas de diretoria com conexão Wi-Fi para controle de estoque de bebidas.

HARDWARE: Generic Wi-Fi Chips
RISK: Entry Point to Boardroom Net
